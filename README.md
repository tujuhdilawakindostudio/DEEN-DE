DEEN DE or Deep Extraculicutter Extra Nonmainstream Dekstop Environtment is a antimainstream desktop for the mainstream laptop,
only for Ubuntu :),

====HOW TO INSTALL====


1. PPA (Recomended)

First, add repo ppa to console or terminal:

sudo apt-add-repository ppa:tujuhdis/deen-desktop

than, update:

sudo apt update

than install:

sudo apt install deen-desktop -y

2. DEB Package

first, install deb from files, than,
open GDebi, install!
